壁纸GET酱

简易壁纸批量获取脚本

一次偶然的机会，我摸到了一个虽然很小，但是有很多动漫壁纸资源的网站并mark了下来，成为了我电脑桌面的主要获取来源。所以，这次的爬虫的目标就是爬取这个网站选取页上的所有缩略图的原图。考虑到此次实现的内容还挺简单的，算是一个日常向的东西。->壁纸来源<-

爬取思路

考虑到爬取的内容为上文网站中的静态资源。因此，爬取的关键在于在网页源码中找到该类型资源在网站布局中的逻辑、规律和所处的标签层级。

壁纸页选择

根据网页URL的特征http://konachan.net/post?page=，不难看出，只要在page=后面跟页码，即可完成页面跳转。

页面元素层级


很明显可以看出壁纸展示区位于<ul id="post-list-posts>...</ul>标签下。每一个缩略图对应一对<li></li>标签。

定位了要探索的主要内容的区域后。下一步，是从缩略图标签中提取出原图所对应的url，即超链接。

根据上述截图，可以看出，图中最下面的<a></a>标签中的url是我们需要的。

实现

为了节省时间（偷懒），选择Python3 + re + requests + bs4作为实现工具，考虑使用最不用脑子的方式————使用单线程和无界面的方式爬取文件。具体过程不做详细解释, 源码如下。

#!/usr/local/bin/python3
'''
' This is spider and batch script which is used for downloading
' wallpapers from particular wallpaper website.
'''
import os, sys
import requests
from bs4 import BeautifulSoup
from bs4.element import *
import re

DIR = './picture/'

def process(page, file_ctr):
    if file_ctr == None:
        file_ctr = 1
    # load html src
    html = requests.get('http://konachan.net/post?page=' + str(page))
    html.encoding = html.apparent_encoding
    html_src = html.text

    # parse html element
    soup = BeautifulSoup(html_src, 'html.parser')
    for i in soup.find_all("ul", id="post-list-posts"):
        for j in i.find_all("li"):
            k = j.find("a", class_=re.compile("directlink \w*img"))
            if k != None:
                img_url = k["href"]
                print(img_url)
                suffix = re.search("\.\w{3,5}$", img_url)
                img_response = requests.get(img_url)
                fd = open(DIR + str(file_ctr) + suffix.group(), 'wb')
                if fd.writable():
                    fd.write(img_response.content)
                fd.close()
                file_ctr = file_ctr + 1


def main():
    # use cmdline parameters
    num_of_params = len(sys.argv)
    start = 1
    end = 1
    if num_of_params == 1:
        print("wrong parameters, append '-h' for help")
        return -1
    elif num_of_params == 2:
        if sys.argv[1] == '-h':
            print("please use './main.py <startpage> <endpage>' cmd.")
            return -1
        else:
            print("wrong parameters, append '-h' for help")
            return -1
    elif num_of_params == 3:
        try:
            start = int(sys.argv[1])
            end = int(sys.argv[2])
        except Exception as e:
            print("wrong parameters, append '-h' for help")
            return -1
    else:
        print("wrong parameters, append '-h' for help")
        return -1

    ctr = 0
    # count file number in picture directory
    for i in os.listdir(DIR):
        ctr = ctr + 1
    for i in range(start, end + 1):
        process(i, ctr)
    return 0


if __name__ == '__main__':
    main()
